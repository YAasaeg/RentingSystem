# 

这是一个面向租客和房东的租房管理系统

## 主要功能

*   **房源管理**
    *   房东可以上传房源信息并就行管理
    *   租客可查看房源
*   **申请管理**
    *   租客可提交租房申请
    *  房东可审核信息后处理申请，通过申请将自动生成合同
*  **聊天功能**
    *租客和房东可进行交流，登录时可查阅聊天记录
*   **预约管理**
    *   租客可预约看房时间
    *   房东可以处理预约信息
